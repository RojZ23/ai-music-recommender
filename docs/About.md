# Documentation – AI Music Recommender System

This folder contains all formal documentation required for the project:

- `srs/` – Software Requirements Specification (combined document with all snapshots).
- `sdd/` – Software Design Document (combined document with all snapshots).
- `design-spec/` – Detailed breakdown of each page, feature, and tool.
- `snapshot-objectives/` – Goals and reflections for start, checkpoints, and final due date.
- `testrail/` – TestRail exports for snapshots 2–4 (test cases and runs).
- `diagrams/` – High-level workflow diagram and related visuals.

Each subfolder also has its own small readme to explain how to use the files inside.
