# Diagrams

This folder contains high-level diagrams for the AI Music Recommender System.

## Files

- `workflow_diagram.drawio`  
  A high-level diagram showing:
  - User interaction with the frontend (UI/UX).
  - Requests from the frontend to backend APIs.
  - How data flows to the database and recommendation engine.
  - Where mood, social, and Quick Mix logic are applied.

You can open and edit this file using diagrams.net / draw.io, and export PNG/PDF for inclusion in reports.
